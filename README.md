# Diabetes-Model
Overview

This project focuses on predicting diabetes-related outcomes using Machine Learning techniques and exploratory data analysis (EDA). The notebook includes data preprocessing, feature engineering, dimensionality reduction, visualization, model training, and performance evaluation using multiple algorithms.

The project uses the Diabetes Dataset from Kaggle and applies classification models to analyze patient-related medical records and predict outcomes effectively.

Features

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Correlation analysis and visualization

Feature transformation and scaling

Principal Component Analysis (PCA)

Decision Tree Classification

Support Vector Machine (SVM)

Model evaluation using accuracy score and confusion matrix

Feature importance and coefficient analysis using Lasso
